# gulp使用
0、gulp serve    启动开发模式(开启一个服务器，跟下面几种gulp无关,更多配置，期待大神配置的方便点)

1、gulp build    切换到生产模式

2、gulp          生产模式下的gulp

3、gulp watch    生产模式下，时时监控（有个小bug，如果开启了监控后，新增文件，gulp监控不到新增文件的代码变化，解决方案，就是关掉监控，重开）

4、gulp release  切换到线上模式
